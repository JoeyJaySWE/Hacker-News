## How to use

1. Clone the repository
```
$ git clone https://github.com/trilisen/hacker-news
```
2. Change your directory to the repository
```
$ cd hacker-news
```
3. Start a localhost server via php
```
$ php -S localhost:8000
```
4. Open your browser
```
localhost:8000
```
